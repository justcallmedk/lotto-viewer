# Lottery Numbers Analyzer

## Demo
https://lotto.sooda.io

## Dependencies
- Node.js
- React
- Express
- MySQL
   - Data Generator (https://github.com/justcallmedk/lotto-generator)

## Getting Started
```bash
$ npm install
$ node express/app.js {db_host} {db_username} {db_password} &
$ npm start
```
